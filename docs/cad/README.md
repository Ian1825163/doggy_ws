# CAD Uploads

Put CAD-related project media here while the Doggy assembly and URDF are still being rebuilt.

Good files to keep here:

- Onshape assembly screenshots.
- Frame and mate connector screenshots.
- Exported STEP/STL snapshots that are useful for review.
- URDF tree screenshots from `check_urdf` or RViz.

Suggested names:

- `doggy-cad-overview.png`
- `doggy-urdf-frames.png`
- `doggy-assembly-notes.md`

Do not put Onshape API keys or private `.env` files in this folder.
