# Doggy ROS 2 Workspace

Doggy is a small quadruped robot project running on ROS 2. This workspace keeps the current control stack, motor communication code, and the Onshape-to-URDF export pipeline in one place.

## Current Status

- `quadruped_control` contains keyboard input, gait generation, inverse kinematics, and the motor serial bridge.
- `quadruped_bringup` launches the control nodes and records the main gait, communication, and IMU topics.
- `doggy` contains the Onshape-generated URDF, mesh assets, and export config.
- The URDF assembly is still being rebuilt. The `RL` leg DOF setup is being used as the first reference before duplicating the remaining legs and frames.
- The calf/knee linkage is nonlinear, so the exact motor-to-knee relation should live in the control or joint-state conversion code instead of a simple URDF gear relation.

## Build

```bash
cd ~/doggy_ws
colcon build --symlink-install
source install/setup.bash
```

## Run

Launch the keyboard, gait generator, motor control node, and rosbag recorder:

```bash
cd ~/doggy_ws
source install/setup.bash
ros2 launch quadruped_bringup bringup.launch.py serial_port:=/dev/ttyACM0 baud_rate:=115200
```

Useful individual nodes:

```bash
ros2 run quadruped_control keyboard_node
ros2 run quadruped_control gait_generator_node
ros2 run quadruped_control motor_control_node
```

## Onshape URDF Export

The Onshape export config lives at `src/doggy/config.json`. The current assembly target is `doggy_ros_export`.

Create `src/.env` from `src/.env.example`, then fill in the Onshape API keys:

```bash
cd ~/doggy_ws/src
cp .env.example .env
```

Do not commit `src/.env`; it contains private API credentials.

Export and validate the URDF:

```bash
cd ~/doggy_ws/src
onshape-to-robot doggy
check_urdf doggy/robot.urdf
```

Onshape naming conventions used by the exporter:

- `dof_<name>` for actuated or moving joints.
- `frame_<name>` for helper frames such as `frame_base_link`, `frame_imu`, and each foot frame.
- Keep the robot body fixed/locked as the root of the export assembly.
